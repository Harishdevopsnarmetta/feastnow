🔧 Firebase Setup
Google login needs Firebase. Follow these 5 steps, then re-upload the file with your config.
1. Go to console.firebase.google.com → New project → Add Web App
2. Authentication → Sign-in method → Enable Google
3. Firestore Database → Create database (Start in test mode)
4. Project Settings → Your apps → Copy config → paste into firebaseConfig in this file:
const firebaseConfig = {
 apiKey: "AIza...",
 authDomain: "your-app.firebaseapp.com",
 projectId: "your-app-id",
 storageBucket: "your-app.appspot.com",
 messagingSenderId: "123456",
 appId: "1:123456:web:abc"
};
5. Firebase Console → Authentication → Settings → Authorized domains → Add harishdevopsnarmetta.github.io
